# StarsRater Component
## Five star icons
Install react-icons: FaStar - filled star, FaRegStar - empty star. FaHalfStarAlt - half filled star<br>
integer part of rating value - filled stars <br>
Fractional part 0.25 - 0.75 - half filled star, less than 0.25 - empty star, greater than 0.75 - filled star<br>
# Metacritic presentation
## Badge Chakra Component
If value of metacritic less than 90 badge should have lightgray background color and black color<bg>
If value of metacritic greater than 89 badge should have green background color and white color
